### Установка приложения

#### Склонируйте репозиторий и установите зависимости.
```sh
$ npm install
```

#### Запуск приложения в режиме разработки
```sh
$ npm start
```

#### Создание "продакшн" сборки
```sh
$ npm run build
```
После выполнения задачи собранный проект появится в папке `build/`
